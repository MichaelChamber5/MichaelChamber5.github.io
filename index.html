Some of my Projects:

Asteroids 3D!

This project was made in unity using C#. I'm pretty proud of this one (made in less than 48 hours!). I think the core mechanic of flying through space and shooting asteroids is pretty fun. I've been told the controls are challenging, but I argue that realistic flight controls are more fun. I had to implement some interesting coding tricks to make the game playable without good gaming hardware. I used an object pool to store the asteroids (rather than contuously instantiating and deleting). I also created a spawning buble around the player with an external despawning bubble. This essentially makes the game infinite, allowing the player to fly as far as they want in any direction, spawning in new asteroids and removing asteroids that are far out of range.

Bombies

This project was made in unity using C#. This was a homework project made for my 3D games class. It's unfinished (as of now). I think the gameplay is still polished/fun enough to show off regardless. If I had the time, I would add player and zombie models to make the game... actually have zombies. I would also work on making a nicer looking menu system. Other than that, the game is at a decent state.

Poker Hand Probability

I wrote a program in java that allows the user to run x number of iterations of random hands. It then follows an algorithm to find the best play from that hand (for example, if a hand was fully hearts, the code would return a value of flush (assuming that a flush is the best play we can make from our hand)).

Image to ASCII

I wrote a program in python that allows the user to upload an image and changes a few settings (such as pixel width and inversion). The program then prints an ASCII image of the image uploaded.

Letterboxed

I wrote a program in java that emulates the Letterboxed game on the NYT website. I also wrote an A* algorithm that solves the puzzle in the least amount of words/letters possible.

Marble Maze

I wrote this game in unity using C#. This was my first 3D games project. It's pretty simple but it was a fun project. I used depth first search to create a proceduraly generated maze.
